Jupiterlight

About

Jupiterlight is a 3rd person space shooter styled after games like Star Fox Assault and StarWars Battlefront 2. The game currently features one type of enemy AI, one playable ship type and two missions: one battle mission and one escape mission.
I have long since dropped this project, but I may continue work on it in the future.
